A small Vite + React + Tailwind demo containing the SAT Math Hub UI.

## Prerequisites
- Node.js 18+ installed
- macOS zsh (commands below assume zsh)

## Install

```bash
cd /Users/juanmartinez-berrios/Desktop/MyWebsite
npm install
```

## Run (development)

```bash
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Notes
- The project uses Tailwind CSS for styling and `lucide-react` for icons.
- If you want to use Yarn or PNPM, adapt the install commands accordingly.
